# SSF-0003 — OTA-Brücke implementieren

## Kontext

SSF-Lernpfade sollen auf OTA-Dokumente verweisen können
(z.B. Primärquellen, Forschungsberichte). Die Brücke
ist in `docs/REQUEST-OTA-BRIDGE-2026-07.md` beschrieben
aber noch nicht implementiert.

## Voraussetzungen

- KG-REQ-20260718-004 abgeschlossen (4 KD-IDs registriert)
- OTA-Batch-Registrierung (204 Dokumente) abgeschlossen

## Gewünschtes Ergebnis

In SSF-Lernpfaden: Verlinkung auf OTA-Dokumente als
"Primärquelle" oder "Weiterführende Lektüre" am Ende
einer Lernreise oder eines Kapitels.

## Technischer Ansatz

```typescript
// In LearningPathUnit:
sources?: {
  ota?: string[];  // OTA-Dokument-IDs

}
```
