# SSF-0001 — domainsNeeded auf KD:* aktualisieren nach KG-Migration

## Kontext

Nach Abschluss von KG-REQ-20260718-001 (KNOW:* → KD:* Migration)
muss SSF die `domainsNeeded`-Felder in `lib/learningPaths.ts`
von `KNOW:*` auf `KD:*` aktualisieren.

## Betroffene Datei

`lib/learningPaths.ts` — alle `domainsNeeded`-Arrays in 50 Pfaden.

## Beispiel

```typescript
// Vorher (KNOW:*)
domainsNeeded: ['KNOW:PHY-OPTICS', 'KNOW:PHY-WAVES'],

// Nachher (KD:*)
domainsNeeded: ['KD:PHY-OPTICS', 'KD:PHY-WAVES'],
```

## Auslöser

Erst nach Freigabe durch KG (KG-REQ-20260718-001 status: done).

## Aufwand

Suche-Ersetze in learningPaths.ts — ca. 150 Vorkommen.
Kein inhaltlicher Eingriff.
