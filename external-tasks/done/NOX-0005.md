## Baumeister-Universum: Titan-Energieversorgung

Titan hat kein Sonnenlicht für Solarenergie (Saturn 9.5 AU entfernt).
Diese Keys ermöglichen die Titan-Energieinfrastruktur:

```
Methan (Titan-Atmosphäre)
    + Sauerstoff (aus Wasserelektrolyse via PHY:PEM)
    = Verbrennung → Energie + CO₂ + H₂O

PHY:PEM → Wasser → H₂ + O₂
TOOL:COMBUSTION → CH₄ + O₂ → Energie
ENV:GREEN-HYDROGEN → H₂-Treibstoff für Raumschiffe

CHEM:IRIDIUM → IrO₂-Anodenkatalysator für PEM auf Titan
ENV:CRITICAL-MATERIALS → Iridium-Recycling-Loop einrichten
TOOL:ENERGY-HARVESTING → Vibrationsenergie aus Titan-Windstürmen
SENSOR:MAGNETIC → Magnetische Anomalien = mögliche Metallerzvorkommen
```

## Mission-Typen die diese Keys ermöglichen

1. **MISSION:HYDROGEN-DEPOT** (PHY:PEM + ENV:GREEN-HYDROGEN)
   Titan als Wasserstoff-Tankstelle für Tiefraummissionen

2. **MISSION:ORE-SCAN** (SENSOR:MAGNETIC + CHEM:PLATINUM-GROUP)
   Planetarische Erzsuche nach Platingruppenmetallen

3. **MISSION:RESOURCE-LOOP** (ENV:CIRCULAR-ECONOMY + CHEM:IRIDIUM)
   Geschlossenen Rohstoffkreislauf auf Station etablieren

4. **MISSION:VIBRATION-HARVEST** (TOOL:ENERGY-HARVESTING + SENSOR:PIEZO)
   Titan-Windenergie durch Piezosysteme ernten
