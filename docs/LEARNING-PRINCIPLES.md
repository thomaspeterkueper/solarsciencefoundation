# LEARNING-PRINCIPLES.md
## Wissenspyramide als didaktisches Strukturprinzip

**Status:** Kanonisch  
**Quelle:** SSF-0001  
**Bezug:** SSF-CORE.md §3.1, OTA-ARC-0006-2026-DE, KG-0004-LEARNING-MODEL.md  

---

## 1. Grundprinzip

Wissen baut aufeinander auf. Diese Architektur ist die unsichtbare Struktur der SSF —
Lernende erleben sie, ohne die technischen Labels zu sehen.

Die fünf Ebenen der epistemischen Pyramide (EP0–EP4) bilden das interne Gerüst:

| Ebene | Internes Label | Lernerfahrung (sichtbar) |
|-------|---------------|--------------------------|
| EP0 | Foundation | „Was habe ich wahrgenommen?" |
| EP1 | Knowledge | „Wie nennt man das?" |
| EP2 | Models | „Warum verhält es sich so?" |
| EP3 | Applications | „Was kann ich damit machen?" |
| EP4 | Narratives | „Wo taucht das in der Welt auf?" |

Der Informationsfluss ist ausschließlich EP0 → EP4.  
EP4-Inhalte (Analogien, Geschichten) definieren keine Grundlagen — sie illustrieren sie.

## 2. Mapping: KG-Ebenen → SSF-Modultypen

| KG-Modultyp (ARC-0006) | SSF-Entsprechung | Sichtbar für Lernende |
|-------------------------|------------------|-----------------------|
| grundmodul (L0–L1) | Beobachtungsanker, Basisbegriff | Einstiegsfragen, erste Konzepte |
| brueckenmodul (L1–L2) | Verbindungsmodul | „Das erklärt, warum…" |
| archivmodul | OTA-Dokument-Brücke | „Jetzt kannst du das lesen" |
| reference (transitional) | Vertiefungsreferenz | Weiterführend, nicht blockierend |

---

## 3. Sichtbarkeitsprinzip für die LearnMap

- Labels wie „L0", „Foundation", „Physik" sind interne Architektur — nicht lernerseitig sichtbar
- Jedes Modul zeigt: welche Module es voraussetzt, welche es freischaltet
- Gesperrte Module erscheinen als **sichtbare Vorschau** — Horizont bleibt erhalten
- Tiefenindikator (logarithmische Formel aus SSF-CORE §2.3) reflektiert implizit die EP-Ebene

---

## 4. Designregel für Modulinhalte

Narrative Inhalte (EP4) in SSF-Modulen **illustrieren** Konzepte.  
Sie **definieren** keine Konzepte.  

Die kanonische Definition kommt aus dem KG (via KXF, EP0–EP2).  
Das Narrativ ist der Weg dorthin, nicht das Ziel.

→ Vollständige Redaktionsregeln: `docs/EDITORIAL.md`

---

## 5. Abgrenzung zu anderen Level-Systemen

Das EP0–EP4-System ist **kollisionsfrei** mit:

- **ARC-0006 L0–L9** = Modul-Komplexität (technische Achse)
- **ARC-0005 N0–N3** = Dokument-Vorwissen (Leseranforderung)
- **EP0–EP4** = Epistemische Pyramide (didaktische Strukturachse, SSF-intern)

Alle drei Achsen koexistieren ohne Überschneidung.
