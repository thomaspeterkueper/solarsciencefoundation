# Solar Science Foundation Roadmap

## Phase 0: Foundation

- Repository structure.
- Mission, architecture, membership, API and roadmap docs.
- First learning module.
- First NOXIA demo unlock endpoint.

## Phase 1: Proof loop

Goal:

```text
Learn module -> complete exercise -> store progress -> expose NOXIA unlock
```

Tasks:

- Add exercise submission endpoint.
- Add local/demo progress store.
- Add player progress API.
- Connect NOXIA to `/api/player/[id]/unlocks`.

## Phase 2: KXF integration

- Define `KXF-0.1` import contract.
- Read KUEPER Knowledge Graph exports.
- Map entities to modules, documents and unlocks.
- Reference kueper.com documents.

## Phase 3: Accounts and membership

- User accounts.
- Free Member role.
- Supporting Member and Donor metadata.
- Co-Author role and contributor attribution.
- Curator/admin workflow.

## Phase 4: Course expansion

- Physics foundations.
- Astronomy foundations.
- Spaceflight foundations.
- Biology and life-support foundations.
- Chemistry and materials foundations.

## Phase 5: Production integration

- Real database.
- Durable progress records.
- Authenticated NOXIA API access.
- Versioned KXF imports.
- Module publication workflow.
