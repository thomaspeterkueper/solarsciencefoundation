# Solar Science Foundation Mission

Solar Science Foundation translates the KUEPER Knowledge Graph into short, understandable and game-compatible learning modules.

The platform connects scientific learning with NOXIA. A user learns a concept, completes exercises, stores progress and unlocks relevant scientific capabilities in the game.

## Mission statement

Solar Science Foundation makes foundational science playable, understandable and reusable across the KUEPER ecosystem.

## System role

```text
KUEPER Knowledge Graph = source of truth
kueper.com              = published knowledge documents
SSF                     = learning, progress, membership and API layer
NOXIA                   = game application and unlock consumer
OTA / KUEPER            = archive, hypotheses, models and fictionalisations
```

## First proof

The first milestone is intentionally small:

```text
A player completes SSF-PHY-1101.
SSF records completion.
NOXIA receives UNL:NOX:orbital-navigation.
```

Only after this loop works should the system expand into courses, accounts, membership, curation and larger knowledge graph synchronisation.
