Wie löst man zwei Bedingungen gleichzeitig?

Zwei Einheiten: Der geometrische Blick — zwei Geraden, ein Schnittpunkt. Und das systematische Verfahren — Gauß-Elimination Schritt für Schritt.

Einheit 1 von 2

Zwei Rohre füllen ein Becken — wie lange braucht jedes allein?

Beispiele und Aufgaben sind optional.
Beobachtung
Alltagsproblem
Rohr A und Rohr B füllen zusammen ein Becken in 3 Stunden. Rohr A allein braucht 2 Stunden länger als Rohr B allein. Wie lange braucht jedes Rohr allein? — Eine Unbekannte reicht nicht. Man braucht zwei Gleichungen für zwei Unbekannte.

Ein lineares GleichungssystemLineares Gleichungssystem (LGS)n Gleichungen mit n Unbekannten, wobei jede Gleichung linear ist (keine x², keine xy, keine sin x). Geometrisch: jede Gleichung ist eine Gerade (2D), eine Ebene (3D) oder eine Hyperebene (nD). Die Lösung ist der gemeinsame Schnittpunkt. (LGS) entsteht immer wenn mehrere Bedingungen gleichzeitig erfüllt sein müssen. Jede Bedingung ist eine Gleichung — jede Gleichung ist eine Gerade im Koordinatensystem.

Das Schöne: Die Lösung sieht man im Bild sofort. Wo sich die beiden Geraden schneiden — dort sind beide Bedingungen gleichzeitig erfüllt. Das ist der Punkt (x, y) der die Lösung ist.

a₁x + b₁y = c₁ a₂x + b₂y = c₂
2×2 LGS: zwei Gleichungen, zwei Unbekannte x und y
Jede Zeile = eine Gerade. Lösung = Schnittpunkt der Geraden.

Schnittpunkt bei (2, 1) — beide Gleichungen erfüllt.

Drei mögliche Situationen: Die Geraden schneiden sich in genau einem Punkt → eindeutige Lösung. Die Geraden sind parallel → keine Lösung (widersprüchliche Bedingungen). Die Geraden sind identisch → unendlich viele Lösungen (eine Bedingung ist überflüssig).

Der Schlüssel ist die DeterminanteDeterminantedet A = a₁b₂ − a₂b₁. Ist det ≠ 0: eindeutige Lösung. Ist det = 0: die Geraden sind parallel (oder identisch) — keine oder unendlich viele Lösungen. Die Determinante sagt sofort ob ein LGS lösbar ist.: det A = a₁b₂ − a₂b₁. Ist sie null, sind die Geraden parallel.

Aufgaben
Aufgabe 1
Zwei Züge starten gleichzeitig aufeinander zu: Zug A mit 120 km/h, Zug B mit 80 km/h, Abstand 500 km. Stelle das LGS auf und löse grafisch: Wann und wo treffen sie sich?
Sei t die Zeit (h) und s die Position von Zug A (km). Zug A: s = 120t. Zug B: s = 500 − 80t. Gleichsetzen: 120t = 500 − 80t → 200t = 500 → t = 2,5 h. Position: s = 120·2,5 = 300 km. Die Grafik zeigt zwei Geraden: s = 120t (steigend) und s = 500−80t (fallend). Schnittpunkt bei t = 2,5, s = 300.
Aufgabe 2
Wann hat ein LGS keine Lösung? Stelle ein Beispiel auf und erkläre geometrisch warum.
Beispiel: x + y = 2 und x + y = 5. Beide Geraden haben dieselbe Steigung (−1), aber verschiedene Achsenabschnitte — sie sind parallel und treffen sich nie. Geometrisch: Parallele Geraden haben keinen Schnittpunkt. Algebraisch: det A = 1·1 − 1·1 = 0 → kein eindeutiger Punkt.
Quiz · Einheit 1
Was bedeutet es geometrisch wenn ein 2×2 LGS eine eindeutige Lösung hat?
Jede Gleichung des LGS beschreibt eine Gerade. Die Lösung ist der Punkt der auf allen Geraden gleichzeitig liegt — also der Schnittpunkt. Bei zwei Geraden in der Ebene: ein Schnittpunkt → eindeutige Lösung, parallel → keine Lösung, identisch → unendlich viele.
Was sagt die Determinante det A = a₁b₂ − a₂b₁ aus?
det A = a₁b₂ − a₂b₁. Wenn det ≠ 0: Die Koeffizientenmatrix ist invertierbar → eindeutige Lösung. Wenn det = 0: Die Geraden sind parallel oder identisch → keine oder unendlich viele Lösungen. Die Determinante ist der schnellste Test ob ein LGS lösbar ist.
Gegeben: 2x + y = 5 und x − y = 1. Was ist die Lösung?
Addition der Gleichungen: (2x+y) + (x−y) = 5+1 → 3x = 6 → x = 2. Einsetzen: 2+y = 5 → y = 1. Probe: 2·2+1 = 5 ✓, 2−1 = 1 ✓. det = 2·(−1) − 1·1 = −3 ≠ 0 → eindeutige Lösung.

Einheit 2 von 2

Wie löst man drei Gleichungen mit drei Unbekannten — ohne zu raten?

Aufgaben sind optional.
Erklärung
Das Prinzip
Bei zwei Unbekannten sieht man den Schnittpunkt im Bild. Bei drei Unbekannten schneiden sich Ebenen — das kann man nicht mehr einfach zeichnen. Gauß-Elimination löst das Problem systematisch: Man formt die Gleichungen so um, bis man eine davon direkt ablesen kann — und setzt rückwärts ein.

Die Idee: Erlaubte Operationen die das LGS nicht verändern — eine Gleichung mit einer Zahl multiplizieren, zwei Gleichungen addieren, zwei Gleichungen tauschen. Durch geschickte Kombination eliminiert man eine Unbekannte nach der anderen.

Das Ziel: Die StufenformZeilenstufenformEine Matrix in Zeilenstufenform hat unter jeder Pivot-Position nur Nullen. Die letzte Zeile hat nur eine Unbekannte — die kann man direkt ablesen. Dann rückwärts einsetzen: von unten nach oben jede Zeile auflösen. — ein Dreieck aus Nullen entsteht. Die unterste Zeile hat nur noch eine Unbekannte, die man direkt abliest. Dann von unten nach oben rückwärts einsetzen.

Gauß-Elimination — Schritt für Schritt
Erkunden
Klicke durch die Schritte der Gauß-Elimination. Beobachte wie in jedem Schritt eine Unbekannte eliminiert wird — bis die Lösung ablesbar ist.
Beispiel LGS
2x + y − z = 8
−3x − y + 2z = −11
−2x + y + 2z = −3
Schritt 0 — Ausgangsmatrix
| 2  1 −1 |  8 |
|−3 −1  2 | −11|
|−2  1  2 |  −3|
Schritt 1 — Pivot: erste Spalte
| 2  1 −1 |  8 |
| 0  1  1 |  1 |
| 0  2  1 |  5 |
Z₂ ← 2·Z₂ + 3·Z₁  ·  Z₃ ← Z₃ + Z₁
Schritt 2 — Pivot: zweite Spalte
| 2  1 −1 |  8 |
| 0  1  1 |  1 |
| 0  0 −1 |  3 |
Z₃ ← Z₃ − 2·Z₂
Schritt 3 — Rückwärtseinsetzen
z = −3  (aus Z₃: −z = 3)
y = 1 − z = 1+3 = 4  (aus Z₂)
x = (8−y+z)/2 = (8−4−3)/2 = 2
Lösung: x = 2, y = 4, z = −3
Schritt 0 von 3
Seitenast — optional
Cramer'sche Regel — Lösung als Quotient von Determinanten
Seitenast · Cramer Cramer'sche Regel

Für das 2×2 LGS a₁x + b₁y = c₁ und a₂x + b₂y = c₂:

det A = a₁b₂ − a₂b₁ x = (c₁b₂ − c₂b₁) / det A y = (a₁c₂ − a₂c₁) / det A
A_x entsteht indem man die x-Spalte durch c ersetzt — analog A_y.
Elegant für 2×2, aufwendig für große Systeme → dann lieber Gauß.

Für 2x + y = 5 und x − y = 1: det A = 2·(−1) − 1·1 = −3. x = (5·(−1) − 1·1)/(−3) = −6/(−3) = 2. y = (2·1 − 1·5)/(−3) = −3/(−3) = 1. Schnell und elegant — aber bei 10 Unbekannten nicht mehr praktikabel. Dann ist Gauß-Elimination effizienter.

Aufgaben
Aufgabe 1
Löse mit Gauß-Elimination: x + y = 4 und 3x − y = 8.
Matrix: [1 1 | 4] und [3 −1 | 8]. Schritt 1: Z₂ ← Z₂ − 3·Z₁: [3−3 −1−3 | 8−12] = [0 −4 | −4]. Aus Z₂: −4y = −4 → y = 1. Rückwärts in Z₁: x + 1 = 4 → x = 3. Lösung: x = 3, y = 1.
Aufgabe 2
In der Statik: Zwei Kräfte F₁ und F₂ halten ein Bauteil im Gleichgewicht. ΣFx = 0: F₁·cos30° + F₂·cos60° = 100 N. ΣFy = 0: F₁·sin30° − F₂·sin60° = 0. Löse das LGS für F₁ und F₂.
cos30° = √3/2 ≈ 0,866, sin30° = 0,5, cos60° = 0,5, sin60° = √3/2 ≈ 0,866. System: 0,866·F₁ + 0,5·F₂ = 100 und 0,5·F₁ − 0,866·F₂ = 0. Aus Gl.2: F₁ = 1,732·F₂. Einsetzen in Gl.1: 0,866·1,732·F₂ + 0,5·F₂ = 100 → 1,5·F₂ + 0,5·F₂ = 100 → F₂ = 50 N. F₁ = 1,732·50 = 86,6 N.
Quiz · Einheit 2
Was ist das Ziel der Gauß-Elimination?
Zeilenstufenform: Unter jeder Pivot-Position stehen nur Nullen — das System sieht aus wie ein Dreieck. Die unterste Zeile hat nur noch eine Unbekannte. Von dort rückwärts einsetzen: jede Zeile hat dann eine Unbekannte mehr, die man aus den bereits bekannten berechnet.
Welche Operationen sind bei der Gauß-Elimination erlaubt — ohne das LGS zu verändern?
Drei erlaubte Operationen (Elementaroperationen): (1) Zeile mit λ ≠ 0 multiplizieren, (2) Vielfaches einer Zeile zu einer anderen addieren, (3) zwei Zeilen tauschen. Diese Operationen ändern die Lösungsmenge nicht — sie formen nur die Darstellung um. Alles andere wäre verboten.
Wann hat das LGS 2x + 4y = 6 und x + 2y = 3 unendlich viele Lösungen?
2x+4y=6 ist dasselbe wie x+2y=3 (mal 2). Beide Gleichungen sind identisch — sie beschreiben dieselbe Gerade. Gauß würde Z₂ ← Z₂ − Z₁/2 = [0 0 | 0] ergeben — eine Nullzeile. Das bedeutet: eine Gleichung ist überflüssig, eine Unbekannte ist frei wählbar (freier Parameter). Unendlich viele Lösungen: x = 3−2t, y = t für beliebiges t.
